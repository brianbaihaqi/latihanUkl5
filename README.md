Program ini dirancang untuk membantu dalam merekap nilai ujian . Pertama masukkan jumlah siswa sebagai input. Kemudian, program menggunakan perulangan for untuk meminta nilai ujian dari setiap siswa satu per satu dan menjumlahkannya ke dalam variabel totalNilai. Setelah semua nilai dimasukkan, program menghitung rata-rata dengan membagi total nilai dengan jumlah siswa, lalu mencetak hasilnya ke layar.
